# Running PHPUnit Tests

To run all tests in this project:

### Clone this repository
```git
git clone https://github.com/itsfirestorm/MP0613_RA1_RA2_IntroPHP.git
cd MP0613_RA1_RA2_IntroPHP
```

## Run the tests:
### Linux / macOS
```bash
./vendor/bin/phpunit ./tests/*
```

### Powershell / CMD
```powershell
.\vendor\bin\phpunit .\tests\*
```
